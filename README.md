# ⚙️ Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/codealpha-tasks.git
```

## 2️⃣ Navigate to Project Folder

```bash
cd codealpha-tasks
```

## 3️⃣ Install Dependencies

For each backend project:

```bash
npm install
```

## 4️⃣ Run Development Server

```bash
npm run dev
```

or

```bash
npm start
```

---

# 🗄 Database Setup

1. Install XAMPP
2. Start Apache & MySQL
3. Open phpMyAdmin
4. Create required databases
5. Import SQL files if included
